# YQLYwxbot

Windows 桌面微信 AI 机器人。

---

## **功能介绍**（YQLYwxbot v5.0.3）

###### 基于 wxautox4 的微信 AI 机器人，支持私聊、群聊、语音通话、朋友圈等全场景智能交互。

### 部分功能列表：

##### **一、AI 对话引擎**

###### **模型与角色**

* 多角色扮演：每个微信好友绑定独立的角色 Prompt，同一机器人同时扮演多个角色，对不同人说不同话
* 多模型兼容：OpenAI API 格式通用接口，可接入云端 API 或本地部署模型
* 本地推理：支持 GGUF 等格式的模型本地运行，无需联网
* 辅助模型分流：可配置独立的辅助模型处理表情判断、联网搜索、定时提醒等轻量任务，主模型专注对话

###### **对话管理**

* 上下文轮数：可配置携带的历史对话轮数，平衡连贯性与成本
* 会话自动切割：检测消息间隔时间，超过阈值自动视为新会话，旧上下文不发送，避免跨天聊天混淆
* 消息队列：并发消息排队处理，自动调节发送间隔，不丢消息
* 失败重试：消息发送、图片识别、表情发送失败自动重试

---

##### **二、感官系统**

###### **视觉**

* 图片理解：多模态视觉模型分析聊天中收到的图片内容，AI 能"看见"图
* 表情包情绪识别：识别对方发送的表情包情绪，据此匹配回复策略
* 表情包库：本地表情包按中文情绪分类（开心、伤心、生气、惊讶、期待、爱慕、尴尬、困惑、疲惫、躲闪等数十种），AI 根据对话情绪从对应目录选发，可配发送概率

###### **听觉与语音**

* 语音识别：流式识别微信语音消息，实时转文字
* AI 语音通话：接听微信语音通话，AI 实时语音对话，支持在聊天中自然发起
* 语音消息发送：直接发送微信原生语音消息
* TTS 合成：文本转语音朗读，支持多种音色
* 音色复刻：上传录音样本克隆专属声音，用于语音通话和消息（CosyVoice + Qwen 双模型）

###### **信息获取**

* 联网搜索：AI 自动判断是否需要联网，实时查询天气、新闻、股价、赛事等时效信息，独立模型配置
* 网页抓取：自动提取消息中链接的网页内容，支持自定义 UA
* AI 图片生成：对话中按需生成图片（兼容 DALL-E 等接口）

---

##### **三、社交互动**

###### **一对一私聊**

* 主动搭话：AI 根据角色人设和近期聊天记忆，在适当时间主动发起对话，可配频率和安静时段
* 自然延迟：回复前模拟阅读思考时间，打字中途随机停顿，短消息快速连发，整体节奏像真人
* 时间感知：AI 知道当前日期、星期、时间，语气和话题随时段自然变化
* **察言观色**：AI 自动判断对话气氛——话题冷了就自然收尾、对方说晚安就暂离、深夜或敷衍时简短回应，不再每条必回。五种状态自动切换：活跃/收尾/暂离/被动/等待
* 拍一拍：识别对方发来的拍一拍，AI 根据角色性格自动回应或反向拍回
* 撤回消息：AI 可撤回自己刚发送的消息，支持随机延迟模拟"犹豫撤回"或"秒撤"，可配撤回冷却
* 引用回复：支持引用聊天中的某条消息进行针对性回复

###### **朋友圈**

* 自动发布：AI 按角色人设自动编写朋友圈文案并发布
* 浏览互动：自动浏览好友朋友圈，择机点赞和评论
* 风控保护：可配每日发布上限、最小间隔、安静时段，模拟人类使用频率

###### **群聊**

* 被 @ 回复：群里被点名时智能应答
* 关键词监听：可设触发关键词和回复概率
* 消息类型全覆盖：文本、图片、视频、文件、位置、名片、语音、合并转发、引用消息，均能识别处理
* 主动消息过滤：可选择不对群聊发送主动消息

---

##### **四、记忆系统**

###### **双层架构**

* CoreMemory（核心记忆）：长期记忆，存储重要事件、用户偏好、关系变化等，持久化 JSON 文件，可在配置器中手动编辑
* Memory Temp（临时记忆）：短期对话上下文，维持当前会话的连贯性

###### **智能管理**

* 重要度评分：综合事件重要度与存在时间的衰减算法，自动淘汰低价值记忆
* 情感加权：高情感密度的事件获得保护分，不易被淘汰
* 启动翻看：程序重启时自动滚动微信窗口翻看近期聊天记录，不因重启丢失上下文
* 手动总结：支持指令触发临时记忆总结，将近期对话提炼为核心记忆

---

##### **五、运维与配置**

###### **Web 控制台**

* 可视化面板：分类标签页管理全部配置项，定制深色主题
* 密码保护：可选登录密码，API Key 自动隐藏
* 导入导出：支持导入旧版配置或完整数据迁移，导入前自动备份
* 运行日志：浏览器实时查看程序日志
* 在线编辑器：直接编辑角色 Prompt 文件
* **一键自检**：43 项全功能自动检测，30 秒出结果
* **一键备份**：配置/记忆/上下文一键打包
* 一键恢复默认

###### **自动化运维**

* 定时重启：按运行时长 + 无活动检测自动重启，防止内存膨胀，重启前自动保存上下文和提醒
* 邮件告警：程序异常时自动 SMTP 邮件通知
* 进程守护：心跳检测，后台进程异常自动告警

###### **安全与合规**

* 内容过滤：敏感词检测，遇到敏感内容自动清除上下文
* 版本兼容：启动时检测微信版本
* 表情校验：过滤 AI 编造的不存在表情，确保消息发送成功
* 消息去重：语音和文本合并发送时智能处理，避免重复

---

##### **六、交互指令**

| 指令 | 功能 |
|-|-|
| 重启 | 重启机器人程序 |

| 清除临时记忆 | 清空当前对话上下文 |
| 总结记忆 | 将近期对话提炼为核心记忆 |
| 发朋友圈 | 手动触发发布朋友圈 |
